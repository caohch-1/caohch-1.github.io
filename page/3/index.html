3.4 Activityзҡ„з”ҹе‘Ҫе‘Ёжңҹ

Activityиў«еӯҳж”ҫеңЁж Ҳдёӯ з§°дёәиҝ”еӣһж Ҳ(back stack)

иҝҗиЎҢзҠ¶жҖҒпјҡдҪҚдәҺж ҲйЎ¶

жҡӮеҒңзҠ¶жҖҒпјҡдёҚеңЁж ҲйЎ¶дҪҶд»Қз„¶еҸҜи§Ғ

еҒңжӯўзҠ¶жҖҒпјҡдёҚеңЁж ҲйЎ¶д№ҹдёҚеҸҜи§Ғ

й”ҖжҜҒзҠ¶жҖҒпјҡдёҚеңЁж Ҳдёӯ

дёғеӨ§Activityеӣһи°ғж–№жі•

onCreate () еңЁActivityз¬¬дёҖж¬Ўиў«еҲӣе»әж—¶и°ғз”Ё

onStart () з”ұдёҚеҸҜи§ҒеҸҳдёәеҸҜи§Ғж—¶и°ғз”Ё

onResume () дҪҚдәҺж ҲйЎ¶ еӨ„дәҺиҝҗиЎҢзҠ¶жҖҒ

onPause () еҮҶеӨҮеҗҜеҠЁжҲ–жҒўеӨҚеҸҰдёҖдёӘActivityж—¶и°ғз”Ё иҜҘж–№жі•жү§иЎҢйҖҹеәҰдёҖе®ҡиҰҒ

onStop () Activityе®Ңе…ЁдёҚеҸҜи§Ғж—¶и°ғз”Ё

onDestory () иў«й”ҖжҜҒеүҚи°ғз”Ё

onRestart () йЎҫеҗҚжҖқд№ү з”ұеҒңжӯўеҸҳдёәиҝҗиЎҢзҠ¶жҖҒж—¶и°ғз”Ё

з”ҹе‘Ҫе‘ЁжңҹеҹәзЎҖ

onCreateе’ҢonDestory д№Ӣй—ҙжҳҜе®Ңж•ҙз”ҹеӯҳжңҹ

onStartе’ҢonStop д№Ӣй—ҙжҳҜеҸҜи§Ғз”ҹеӯҳжңҹ

onResumeе’ҢonPause д№Ӣй—ҙжҳҜеүҚеҸ°з”ҹеӯҳжңҹ

Activityиў«еӣһж”¶дәҶжҖҺд№ҲеҠһ

еӣһж”¶еүҚеҸҜд»Ҙдҝқз•ҷдёҙж—¶ж•°жҚ®

onSaveInstanceState ( ) жҗәеёҰдёҖдёӘBundleзұ»еһӢеҸӮж•° з”ЁдәҺдҝқеӯҳж•°жҚ®
