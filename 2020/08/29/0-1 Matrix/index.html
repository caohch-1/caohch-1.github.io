0-1 Matrix

Posted on 2020-08-29 In LeetCode

LeetCode:542 0-1矩阵

给定一个由 0 和 1 组成的矩阵，找出每个元素到最近的 0 的距离

两个相邻元素间的距离为 1

i.e.1

i.e.2

注意:

  1. 给定矩阵的元素个数不超过 10000
  2. 给定矩阵中至少有一个元素是 0
  3. 矩阵中的元素只在四个方向上相邻: 上、下、左、右

答案

一下子就能想到是BFS，但是因为一开始的BFS思路和答案的BFS思路不一样，所以记录一下

方案1:

一开始的思路以1的元素为中心，向外检索寻找最近的0

这种方法会TLE，于是去看了题解

方案2:

以0元素为中心向外检索，改变其他元素到0的距离

感觉一样是BFS，别人就能想到以0为中心检索的比较好的方案，我第一反应就是比较差的方案1为中心检索，感觉自己老笨逼了
